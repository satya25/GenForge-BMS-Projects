# Project Viveka – Onboarding & Training Manual

## 1. Introduction
This manual provides onboarding instructions and training guidelines for faculty, exam coordinators, and administrators using Project Viveka. It ensures smooth adoption of automated grading, evaluation workflows, and secure archival of evaluated papers.

## 2. Audience
- **Faculty:** Perform subjective evaluations, monitor grading consistency.
- **Exam Coordinators:** Track evaluation progress and timelines.
- **Administrators:** Oversee grading workflows, ensure compliance and archival.

## 3. Prerequisites
- Valid institutional login credentials.
- Role-based access assigned by administrator.
- Basic familiarity with exam papers and evaluation processes.

## 4. Onboarding Steps
1. **Login:** Access Viveka via institutional portal with secure credentials.
2. **Role Verification:** Confirm dashboard access matches assigned role.
3. **Profile Setup:** Faculty configure evaluation preferences; coordinators set workflow parameters.
4. **Initial Training:** Attend orientation session (1 hour) covering grading engine and dashboards.

## 5. Training Modules
- **Module 1: Automated Grading**
  - Run grading engine for objective questions.
  - Review accuracy metrics.
  - Validate flagged discrepancies.

- **Module 2: Evaluation Workflow**
  - Faculty complete subjective evaluations.
  - Coordinators track progress and deadlines.
  - Administrators monitor workflow compliance.

- **Module 3: Consistency Checker**
  - Review grading uniformity across evaluators.
  - Resolve flagged inconsistencies.
  - Export consistency reports.

- **Module 4: Integration Awareness**
  - Understand paper flow from Yojanā.
  - Review archival in Sangrah.
  - Access analytics in Anvesha.

## 6. FAQs
- **Q:** How accurate is automated grading?  
  **A:** ≥95% accuracy for objective questions; faculty validate subjective answers.  
- **Q:** Can coordinators override grading timelines?  
  **A:** Yes, coordinators can adjust deadlines in workflow manager.  
- **Q:** How are evaluated papers archived?  
  **A:** Automatically pushed into Sangrah after finalization.

## 7. Troubleshooting
- **Issue:** Login failure → Contact IT admin for credential reset.  
- **Issue:** Grading engine not running → Verify paper upload status.  
- **Issue:** Dashboard not updating → Refresh session or contact support.  

## 8. Training Schedule
- Orientation: Week 1 (all roles).
- Faculty grading workshop: Week 2.
- Coordinator workflow deep-dive: Week 3.
- Administrator compliance session: Week 4.

## 9. Support Channels
- Helpdesk email: support@institute.edu
- Dedicated Slack/Teams channel for queries.
- Monthly Q&A sessions with system admins.

## 10. Continuous Learning
- Quarterly refresher workshops.
- Updated training materials with each release.
- Feedback loop for improving onboarding experience.
